DIR360

Diagnosis of Impulse Response in 360

Contact us Brochure

Features:

  • Acoustic diagnosis by spatial and temporal visualization of impulse responses.
  • Segregation of direct sound and subsequent reflections of a room.
  • Ease detection of acoustical problems.
  • Useful for architectural acoustics, audio production and industrialnoise.

Applications:

  • The advantage in the comprehension and characterization of the acoustic behavior of a room gets improved with the 3D impulse response measurements.
  • Obtaining the signal direction allows us to visualize the temporal distribution characteristics of the room reflections in space, observe the directional distribution of early and late energy, and identify areas that generate unwanted reflections.
  • Moreover, the comparison of the spatial components of impulse responses of various rooms can contribute to identify differences and similarities in acoustic characteristics.
  • This acoustical software provides a complete set of techniques for carrying out an assessment from different perspectives.

Toolbar

  • Python: Native options of the window.
  • File: Allows you to clean the project window, save the measurement set in a project or open a pre-existing one.
  • Import: You can import an image from the plant view to be used in the plant graphic flap (See Figure 12), import an audio file (wav) in format B or in format A. It allows the import of a single multichannel audio file or four mono files.
  • Export: Through Plotly, you export the three-dimensional graphic in an interactive HTML file. The visualization allows to rotate the view to visualize the IR. It also supports exporting the data to an Excel file.

Configuration tab

Windowing
  • Overlapping: Mode "allowed full overlap (Full)" between windows or "allowed overlap of half window (Half)".
  • Windows size: Size of the integration window.
  • Quantity: Number of windows considered.
  • Threshold: Limit the difference in the maximum sound pressure level that a window can have with direct sound.
Audio Editing
  • Audio Cutter: Number of milliseconds of analysis.
  • Filtering: Optional application of low pass and high pass linear phase filters, and their cutting frequencies. Typically a low pass at 5 kHz is used to avoid loss of coherence.
  • Conversion type: It allows to choose the type of conversion to sound intensity between the instantaneous mode and through the application of the FFT.

Direct Sound Selection tab

Finally, the direct sound selection tab allows you to choose which of the first windows represents the direct sound. For this pre-selection, the highest level peaks are obtained and interpreted as the center of the direct sound. For a first instance of calculation, the first one is taken in a temporal order. After the calculation is made, the user can choose to modify the direct sound chosen automatically by one of the others and perform the calculations again.

2D graphics panel

A plant view allows you to evaluate several IRs at the same time. For this it is necessary to have a plant view of the measurements. The program generates a top view of each IR and then place them manually in the plant view.

In the last tab, the calculated values of each window are shown in time, magnitude, azimuth and elevation.

3D graphics panel

The information are represented as sound vectors with the same origin and different direction. The vectors represents each of the time windows. Its magnitude is given by the energy sum the intensity vectors. The magnitudes are normalized based on the direct sound. The color symbolizes the temporal distance to the direct sound (red to blue).

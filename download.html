Steps to download and install

1) Download and install the latest version of Canopy for python 2.7

2) Open Canopy app and go to Package Manager

Download and install this two packages

  • pyqt (Version 4.12)
  • mayavi (Version 4.4.3-10)
  • Please pay attention to the version. By default, Mayavi comes with a newer version, but should be downgraded to version 4.4.10

3) Download IRV360

Windows

Mac OS X

4) Open IRV360 by clicking in IRV360.command (OSX) or IRV360.bat (Windows)
